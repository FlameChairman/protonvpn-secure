# ProtonVPN

### What is ProtonVPN?

ProtonVPN is a privacy-focused VPN service built by the team behind ProtonMail. Based in Switzerland, it operates under some of the strongest data protection laws in the world and provides secure, encrypted internet access with full transparency through its open-source applications.

### Why ProtonVPN?

ProtonVPN routes traffic through Secure Core servers in privacy-friendly countries. It offers a free tier with no data limits, no advertisements, and no logging. All apps are independently audited and published as open source.
